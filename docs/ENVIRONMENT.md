# Environment Variables

See `.env.example` and `.env.staging.example`.

Key variables:
- `DATABASE_URL`
- `JWT_SECRET`
- `JWT_EXPIRES_IN`
- `REFRESH_TOKEN_TTL_DAYS`
- `RATE_LIMIT_WINDOW_MS`
- `RATE_LIMIT_MAX`
- `REDIS_URL` (optional for health checks)
- `SENTRY_DSN` (optional)
- `DATA_RETENTION_DAYS`
